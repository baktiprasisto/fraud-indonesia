# 🛡️ AI Fraud Detection Indonesia

A machine learning-based web application to detect digital fraud such as **Email Spam**, **SMS Scam**, and **Phishing URLs**.  
This project is deployed using **Streamlit** to provide a simple and interactive interface for users.

---

## 📌 Overview

With the rapid growth of digital communication, fraud cases in Indonesia are increasing.  
This project aims to help users identify potential threats using **Artificial Intelligence (AI)**.

The system supports:

- 📧 Email Scam Detection  
- 💬 SMS Spam Detection  
- 🔗 Phishing URL Detection  

---

## 📦 Dataset

This project uses multiple datasets from Kaggle:

### 📧 Email Spam Dataset
🔗 https://www.kaggle.com/datasets/gevabriel/indonesian-email-spam  

### 💬 SMS Spam Dataset
🔗 https://www.kaggle.com/datasets/gevabriel/indonesian-sms-spam  

### 🔗 Phishing URL Dataset
🔗 https://www.kaggle.com/datasets/sai10py/phishing-websites-data  

---

## 🧠 Machine Learning Models

Each detection system uses a different approach:

| Model | Method |
|------|--------|
| Email Spam | TF-IDF + Naive Bayes |
| SMS Spam | TF-IDF + Naive Bayes |
| Phishing URL | Random Forest |
